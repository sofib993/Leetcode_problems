# 🚀 DSA Problems

A personal repository where I document my journey through **Data Structures and Algorithms (DSA)** by solving problems from various competitive programming platforms.

The goal of this repository is to stay consistent, improve problem-solving skills, and prepare for coding interviews and programming contests.

---

## 📚 Platforms

Problems are solved from:

- 🟢 LeetCode
- 🟠 HackerRank
- 🔵 Codeforces

---

## 📂 Repository Structure

Each problem has its own folder containing:

- `solution.py` - Python solution
- `README.md` - Problem description, examples, and explanation (when applicable)

Example:

```text
dsa_problems/
│
├── 9-palindrome-number/
│   ├── solution.py
│   └── README.md
│
├── 14-longest-common-prefix/
│   ├── solution.py
│   └── README.md
│
├── arithmetic-operator/
│   ├── solution.py
│   └── README.md
│
└── ...
```
