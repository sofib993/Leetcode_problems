# Arithmetic Operators

## Task
The provided code stub reads two integers, `a` and `b`, from STDIN. Add code to print three lines where:
1. The first line contains the sum of the two numbers (`a + b`).
2. The second line contains the difference of the two numbers (first - second) (`a - b`).
3. The third line contains the product of the two numbers (`a * b`).

---

## Example
If `a = 3` and `b = 5`:

**Output:**
```text
8
-2
15
```
## Input Format
The first line contains the first integer, `a`.  
The second line contains the second integer, `b`.

## Constraints
* 1 &le; a &le; 10^10
* 1 &le; b &le; 10^10

## Output Format
Print the three lines as explained in the task.
